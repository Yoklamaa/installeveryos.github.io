Windows 11 Kurulumu (UEFI)

Bu rehberde Windows 11'i USB bellek ile nasıl kuracağınızı öğreneceksiniz.

Gerekenler

  • 8 GB veya daha büyük USB bellek
  • Rufus programı (rufus.ie)
  • Windows 11 ISO dosyası (resmi Microsoft sitesinden)

Kurulum Adımları

  1. Rufus ile ISO dosyasını USB'ye yazın
  2. Bilgisayarınızı USB'den başlatın (BIOS/UEFI'den Boot ayarları yapın)
  3. "Install Now" butonuna tıklayın
  4. Disk bölümlendirmeyi yapın ve kurulumu başlatın
