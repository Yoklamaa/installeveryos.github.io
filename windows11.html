Windows 11 Kurulumu (UEFI)

Windows 11’i adım adım kurmayı bu rehberde öğreneceksin:

1. Gerekli Araçlar

  • Windows 11 ISO
  • Rufus Programı
  • En az 8 GB USB bellek

2. USB Hazırlama (Rufus)

  1. Rufus'u çalıştır
  2. ISO'yu seç
  3. Partition scheme: GPT
  4. Target system: UEFI
  5. “BAŞLAT” tıkla

3. BIOS Ayarları

  • BIOS’a gir (DEL veya F2)
  • Secure Boot: Disabled
  • Boot Order: USB en üstte

4. Kurulum Süreci

  1. USB’den başlat
  2. "Install Now" tıkla
  3. Ürün anahtarı atla → sürüm seç
  4. Custom Install seç
  5. Diskleri sil → yeni bölüm oluştur
  6. Kurulumu başlat
